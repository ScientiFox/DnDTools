D&D Tools



A collection of useful tools for doing D&D things!
  • A die-roll parser with an additional test function for checking the results of roll results, especially those including empowerment or re-roll options.
  • Includes a minor magic item creation tool and a character background creator, with properties derived from several sources, as well a quirks.
  • A character background creator derived from the 5e tables!
  • A combat manager that tracks HP, AC, turn-based conditions, and lets you roll dies easily. Each combatant has an amount box that sets healing, damage, or condition duration with the buttons. Clicking the blue atk roll rolls those dice, and whenever there's a roll, it loads the result into the combatants' amount box for easy updates. Click a condition timer to remove it
  • An empowered roll tester, which runs 10k trials to estimate the effect of re-rolls. ndK style tests, with up to mod replacements on each
